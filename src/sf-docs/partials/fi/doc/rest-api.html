REST API (JSON-LD)

Lainsäädäntö

Listan säädöksistä antovuoden (YYYY) mukaan voi hakea kutsulla:

http://data.finlex.fi/api/v1/statutes/(YYYY)

Yksittäisen säädöksen (säädösnumero #) voi hakea kutsulla:

http://data.finlex.fi/api/v1/statutes/(YYYY)/(#)

Lisäksi kutsu tukee seuraavia parametrejä

Parametri Arvo Selitys
time YYYYMMDD voimassa oleva versio ajanhetkellä
language fi, sv sisällön kieli
content txt, html, xml sisällön muoto
pretty - palauttaa JSON-rakenteen luettavammassa muodossa
tree - asiakirjan sisältö ja metadata puumaisena rakenteena
free - kutsun rajoittaminen CC-BY-lisenssillä julkaistuun dataan
  • Esimerkkejä
    • http://data.finlex.fi/api/v1/statutes/2004/1430?pretty
    • http://data.finlex.fi/api/v1/statutes/2004/1430?pretty&tree

Oikeuskäytäntö

Yksittäisen oikeuden ratkaisun voi hakea kutsulla:

http://data.finlex.fi/api/v1/judgements/ECLI:FI:(KHO|KKO):(YYYY):(#)

Lisäksi kutsu tukee seuraavia parametrejä

Parametri Arvo Selitys
language fi, sv sisällön kieli
content txt, html, xml sisällön muoto
  • Esimerkkejä
    • http://data.finlex.fi/api/v1/judgements/ECLI:FI:KHO:2011:50
    • http://data.finlex.fi/api/v1/judgements/ECLI:FI:KKO:2011:100?content=html&language=sv
