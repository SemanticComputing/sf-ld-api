Identifiers: URI practices in the Semantic Finlex service

One of the key publication principles of Linked Data is to provide unique URI identifiers for referable information. The user can search for related information by typing this identifier in the browser or using it programmatically through the HTTP protocol. This page describes how the identifiers used in the Semantic Finlex service (data.finlex.fi) have been created and can be used.

The legal documents

Legislation

The URI template for the statutes is based on the ELI standard (European Legislation Identifier).

URI Description Example
http://data.finlex.fi/eli/sd List of the statutes
http://data.finlex.fi/eli/sd/{year} List of the statutes per year Example
http://data.finlex.fi/eli/sd/{year}/{id} A specific statute Example
http://data.finlex.fi/eli/sd/{year}/{id}/alkup The original version of a specific statute Example
http://data.finlex.fi/eli/sd/{year}/{id}/alkup/(fin|swe)/(html|txt) The original version of a specific statute in Finnish/Swedish Example
http://data.finlex.fi/eli/sd/{year}/{id}/alkup/(fin|swe)/(html|txt|xml) The HTML/XML/text content of the original version of a specific statute Example
http://data.finlex.fi/eli/sd/{year}/{id}
/ajantasa/{YYYYMMDD}		 A consolidated version of a statute at a specific point in time (YYYYMMDD) Example
http://data.finlex.fi/eli/sd/{year}/{id}
/ajantasa/{YYYYMMDD}/(fin|swe)		 A consolidated version of a statute at a specific point in time (YYYYMMDD) in Finnish/Swedish Example
http://data.finlex.fi/eli/sd/{year}/{id}
/ajantasa/{YYYYMMDD}/(fin|swe)/(html|txt)		 The HTML/text content of a consolidated version of a statute at a specific point in time (YYYYMMDD) in Finnish/Swedish Example
http://data.finlex.fi/eli/sd/{year}/{id}
/osa/{#}/luku/{#}/pykala/{#}/...		 A statute item Example
http://data.finlex.fi/eli/sd/{year}/{id}
/osa/{#}/luku/{#}/.../alkup		 The original version of a statute item Example
http://data.finlex.fi/eli/sd/{year}/{id}
/osa/{#}/luku/{#}/.../ajantasa/YYYYMMDD		 A condolidated version of a statute item Example

Case law

The URI template for the judgments is based on the ECLI standard (European Case Law Identifier), but modified to align with the HTTP URI patterns of the ELI standard.

URI Description Example
http://data.finlex.fi/oikeus List of judgments
http://data.finlex.fi/ecli/{court)/{year}/{id} A specific judgment Example
http://data.finlex.fi/ecli/{court)/{year}/{id}/{language} A judgment in Finnish/Swedish Example
http://data.finlex.fi/ecli/{court)/{year}/{id}/{language}/{format} The HTML/text content of a judgment in Finnish/Swedish Example

RDF graphs

Legislation

Statutory law has been divided into two different graphs:

URI Description
http://data.finlex.fi/eli/sd/alkup Contains the original versions of the statutes
http://data.finlex.fi/eli/sd/ajantasa Contains the consolidated (amended) versions of the statutes (Note: CC-BY-NC license)

Case law

The judgments of the Supreme Court and Supreme Administrative Court have been divided into two corresponding graphs:

URI Description
http://data.finlex.fi/oikeus/kho Judgments of the Supreme Administrative Court
http://data.finlex.fi/oikeus/kko Judgments of the Supreme Court

Common resources:

Resources shared by different datasets have been placed into their own graph (http://data.finlex.fi/common). The graph contains for example RDF resources representing the Parliament and the Ministry of Justice.

Vocabularies

Different vocabularies used for subject indexing have been placed into their own graphs:

  • Finlex vocabulary: http://data.finlex.fi/voc/finlex
  • Supreme Court vocabulary: http://data.finlex.fi/voc/kko
  • Supreme Administrative Court vocabulary: http://data.finlex.fi/voc/kho

RDF schemas

  • SFL (Semantic Finlex Legislation): http://data.finlex.fi/schema/sfl
  • SFCL (Semantic Finlex Case Law): http://data.finlex.fi/schema/sfcl
  • SFL (Semantic Finlex Legislation): http://data.finlex.fi/schema/common
