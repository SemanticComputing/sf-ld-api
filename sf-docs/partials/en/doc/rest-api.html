REST API (JSON-LD)

Legislation

List statutes by year (YYYY:

http://data.finlex.fi/api/v1/statutes/(YYYY)

Find a statute by year (YYYY) and id (#) :

http://data.finlex.fi/api/v1/statutes/(YYYY)/(#)

The request supports following parameters:

Parameter Value Description
time YYYYMMDD point in time
language fi, sv language of the content
content txt, html, xml format of the content
pretty - returns JSON in a more readable format
tree - the document content is presented as a hierarchic tree
free - all data returned by the request is available under CC-BY license
  • Examples
    • http://data.finlex.fi/api/v1/statutes/2004/1430?pretty
    • http://data.finlex.fi/api/v1/statutes/2004/1430?pretty&tree

Case law

Find a specific statute by ECLI:

http://data.finlex.fi/api/v1/judgements/ECLI:FI:(KHO|KKO):(YYYY):(#)

The request supports following parameters

Parameter Value Description
language fi, sv language of the content
content txt, html, xml format of the content
  • Examples
    • http://data.finlex.fi/api/v1/judgements/ECLI:FI:KHO:2011:50
    • http://data.finlex.fi/api/v1/judgements/ECLI:FI:KKO:2011:100?content=html&language=sv
