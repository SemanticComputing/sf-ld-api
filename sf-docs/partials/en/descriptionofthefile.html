Purpose of the processing of personal data

The main purpose of the processing of personal data is to collect feedback on the use of Semantic Finlex and to brief users on matters related to the service.

Data content of the file

The file may contain the following information obtained from the customer: first name, last name, name of business/organisation, contact details.

No information will be disclosed outside the service providers (Aalto University, the Ministry of Justice, Edita Publishing Ltd).

Data file protection principles

The personal data file is protected from access by outsiders. Only service providers holding the required user IDs and passwords have access to the system.
